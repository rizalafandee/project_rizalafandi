# 📰 Buletin Statistika Ofisial: Kupas Tuntas IPM Jawa Timur

## 📌 Project Overview
Proyek ini merupakan *Final Project* untuk mata kuliah **Statistika Ofisial D** di **Statistika ITS**. Output utamanya adalah sebuah buletin statistik berjudul **"Kupas Tuntas IPM Jawa Timur: Strategi Akselerasi SDGs untuk Pembangunan Berkelanjutan"**.

Buletin ini menyajikan analisis mendalam mengenai capaian Indeks Pembangunan Manusia (IPM) di Provinsi Jawa Timur serta hubungannya dengan target-target *Sustainable Development Goals* (SDGs).

## ❓ Problem Statement
Pembangunan manusia merupakan indikator keberhasilan jangka panjang suatu daerah. Tantangan utama yang dibahas adalah:
* **Disparitas Wilayah:** Bagaimana sebaran capaian IPM di berbagai kabupaten/kota di Jawa Timur?
* **Akselerasi SDGs:** Sejauh mana komponen pendukung seperti kesehatan, pendidikan, dan pengeluaran per kapita berkontribusi terhadap target SDGs?
* **Strategi Kebijakan:** Langkah apa yang perlu diambil pemerintah daerah untuk meningkatkan kualitas hidup masyarakat secara merata.

## 📂 Content Scope
Buletin ini mencakup analisis komprehensif pada beberapa indikator utama:
1. **Komponen Utama IPM:** Rata-rata Lama Sekolah (RLS), Angka Harapan Hidup (AHH), dan Pengeluaran per Kapita.
2. **Komponen Pendukung:** Angka Melek Huruf (AMH) dan persentase penduduk miskin.
3. **Analisis Sanitasi:** Hubungan antara sanitasi layak dengan kesehatan masyarakat.
4. **Strategi Akselerasi:** Solusi praktis untuk meningkatkan kualitas pendidikan dan kesehatan di Jawa Timur.

## ⚙️ Methodology
* **Data Sourcing:** Menggunakan data resmi dari publikasi Badan Pusat Statistik (BPS) Provinsi Jawa Timur tahun 2024.
* **Analysis:** Penerapan analisis statistik deskriptif dan visualisasi data untuk memetakan capaian IPM antar wilayah.
* **Standard:** Mengacu pada metodologi standar BPS dalam penghitungan komponen-komponen IPM.
