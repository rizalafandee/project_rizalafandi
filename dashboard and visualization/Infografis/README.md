# 🎨 Infographics Gallery

## 📌 Project Overview
Folder ini berisi kumpulan karya infografis yang saya kembangkan untuk menerjemahkan data statistik yang kompleks menjadi informasi visual yang mudah dipahami (*actionable insights*). Koleksi ini mencakup berbagai topik, mulai dari tugas akademik di **Statistika ITS** hingga proyek analisis mandiri.

Setiap infografis dirancang dengan memperhatikan prinsip desain komunikasi visual dan akurasi data statistik.

## ❓ Why Visualization Matters?
Sebagai seorang *Data Analyst*, kemampuan untuk menyampaikan temuan data kepada audiens non-teknis sangatlah krusial:
* **Clarity:** Menyederhanakan tren data yang rumit.
* **Engagement:** Meningkatkan daya tarik informasi melalui elemen visual yang estetis.
* **Efficiency:** Mempercepat proses pengambilan keputusan melalui ringkasan visual yang padat.

## 📂 Content Gallery
Di dalam folder ini, kamu akan menemukan infografis yang mencakup tema:
1. **Analisis Deskriptif:** Visualisasi distribusi dan tren data demografis atau ekonomi.
2. **Business Insight:** Ringkasan eksekutif dari proyek analisis bisnis (seperti perilaku pelanggan atau *churn analysis*).
3. **Statistika Publik:** Edukasi mengenai konsep-konsep statistik untuk khalayak umum.

## 🛠️ Tools Used
Dalam proses pembuatannya, saya menggunakan kombinasi *tools* berikut:
* **Canva:** Untuk penyusunan tata letak dan desain grafis yang modern.
